Select `Menu Bar -> File -> Open Executable` (shortcut `ctrl+e`).

Navigate to the binary you want to open.

## Unloading / stopping a binary (so you can load a different one):

Select `Menu Bar -> Debug -> Stop Debugging` (shortcut shift+F5).

or

Enter `q` in the command window, to **q**uit the process.

[MS Documentation for q](https://docs.microsoft.com/en-us/windows-hardware/drivers/debugger/q--qq--quit-)
