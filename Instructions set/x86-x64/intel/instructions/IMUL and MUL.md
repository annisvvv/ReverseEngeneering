IMUL is fore signed multiply where as MUL is unsigned multiply.
[[How to know if a hex is negative or positive]]
### Forms
- imul r/mX                                    ex:   rdx:rax = rax * r/mX
- imul reg, r/mX                             ex:   reg = reg * r/mX
- imul reg, r/mX, immediate          ex:   reg = r/mX * immediate
