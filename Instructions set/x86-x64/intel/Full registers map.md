## **General-Purpose Registers**

```
┌──────────────────────────────┐
│           RAX (64-bit)       │
├──────────────────────────────┤
│           EAX (32-bit)       │
├──────────────────────┬───────┤
│         AX (16-bit)  │       │
├───────────┬──────────┤       │
│   AH (8)  │  AL (8)   │       │
└───────────┴──────────┴───────┘

┌──────────────────────────────┐
│           RBX                │
├──────────────────────────────┤
│           EBX                │
├──────────────────────┬───────┤
│         BX           │
├───────────┬──────────┤
│   BH      │   BL      │
└───────────┴──────────┘

┌──────────────────────────────┐
│           RCX                │
├──────────────────────────────┤
│           ECX                │
├──────────────────────┬───────┤
│         CX           │
├───────────┬──────────┤
│   CH      │   CL      │
└───────────┴──────────┘

┌──────────────────────────────┐
│           RDX                │
├──────────────────────────────┤
│           EDX                │
├──────────────────────┬───────┤
│         DX           │
├───────────┬──────────┤
│   DH      │   DL      │
└───────────┴──────────┘

```

## **Registers RBP, RSP, RSI, RDI, R8–R15**

These **do not have AH/LH style** 8-bit halves. Only 8-bit low registers exist.

```
┌──────────────────────────────┐
│           RBP                │
├──────────────────────────────┤
│           EBP                │
├──────────────────────────────┤
│            BP                │
└─────────────────┬────────────┘
                  │ BPL (8-bit)

┌──────────────────────────────┐
│           RSP                │
├──────────────────────────────┤
│           ESP                │
├──────────────────────────────┤
│            SP                │
└─────────────────┬────────────┘
                  │ SPL (8-bit)

```

## **Registers R8–R15**

```
┌──────────────────────────────┐
│           R8  64-bit         │
├──────────────────────────────┤
│           R8D 32-bit         │
├──────────────────────────────┤
│           R8W 16-bit         │
└──────────────────────────────┬┘
                               │ R8B  8-bit

┌──────────────────────────────┐
│           R9                 │
├──────────────────────────────┤
│           R9D                │
├──────────────────────────────┤
│           R9W                │
└──────────────────────────────┬┘
                               │ R9B

```

…and the same for **R10–R15**:
- R10 / R10D / R10W / R10B
- R11 / R11D / R11W / R11B
- R12 / R12D / R12W / R12B
- R13 / R13D / R13W / R13B
- R14 / R14D / R14W / R14B
- R15 / R15D / R15W / R15B
